# Office Lite (Rust + Tauri)

A minimal Microsoft Word–like editor with DOCX support.

## Features
- Open .docx
- Edit text
- Save .docx (Office 365 compatible)

## Run

### Backend
cd src-tauri
cargo tauri dev

### Frontend
cd frontend
npm install
npm run dev

## Notes
- Only basic text supported
- Formatting coming in next version
